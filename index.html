QuickScribe - תמלול אודיו קל ופשוט

איך זה עובד?

  1. בחרו קובץ אודיו מהמחשב שלכם
  2. בחרו כמה מילים תרצו בכל כתובית
  3. לחצו על כפתור "תמלול" והמתינו לתוצאות
  4. צפו בתוצאות בפורמט SRT או כטקסט רגיל
  5. העתיקו או הורידו את התמלול לשימוש עתידי

הגבלות:

  • סוגי קבצים נתמכים: MP3, MP4, MPEG, MPGA, M4A, WAV, WEBM
  • גודל קובץ מקסימלי: ללא הגבלה (קבצים גדולים יחולקו אוטומטית)
×

כיצד ליצור API KEY?

  1. גש לאתר console.groq.com והתחבר או צור חשבון חדש.
  2. נווט לאזור "API Keys" בתפריט.
  3. לחץ על "Create API Key" או "Create New API Key".
  4. תן שם למפתח החדש ולחץ על כפתור האישור.
  5. העתק את המפתח שנוצר ושמור אותו במקום בטוח.
  6. הזן את המפתח שהעתקת באתר שלנו.

חשוב: שמור על סודיות המפתח ואל תשתף אותו עם אף אחד.
